# Yoursoft Digital Landing Page

Production-ready multi-page React and Tailwind website for Yoursoft Digital, a Canada-based IT services and SaaS product company.

## Highlights

- Service pages for web development, app development, digital marketing, and graphic design
- Product pages for Zodo CRM, ATS System, and AI Chatbot
- Responsive sticky navigation with service and product dropdowns
- Animated premium homepage hero and conversion-focused sections
- Pricing, about, and contact pages with lead-generation CTAs

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
